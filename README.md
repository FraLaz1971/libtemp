# libtemp
template example c project with a library and many targets
###
To compile on linux
###
`make`
###
To run on linux
`./mfile1 tmdb.txt`
###
To compile on MS Windows with MSVC, open cmd terminal and enter the folder libtemp
###
`nmake /f makefile.vc`
###
To run on windows
###
`mfile1 tmdb.txt`
###
